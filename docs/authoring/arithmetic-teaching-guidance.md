# Arithmetic Teaching Guidance

Use this guide when creating or revising arithmetic pathways, stages, skills, materials, or playlists.

The goal is not to imitate a country's curriculum brand. The goal is to teach arithmetic in the clearest, most durable way for the learner.

## Short Answer

Use a direct, explicit, practice-first style.

That means:

- teach the idea clearly
- show a worked example
- give deliberate practice
- revisit weak spots quickly
- keep cumulative review alive
- build fluency without hiding the underlying structure

## What To Prefer

- explicit instruction for new content
- concise explanation tied to a real arithmetic job
- worked examples before independent practice
- deliberate practice in small sets before mixed review
- cumulative review after new teaching
- fast error correction and reteaching when a weak pattern appears
- fact fluency that supports later written arithmetic
- place-value reasoning before or alongside written methods
- standard written methods once the learner can explain what the digits represent

## What To Avoid

- long discovery-first sequences before the learner has seen a clear explanation
- abstract theory that does not help the learner answer or explain arithmetic more accurately
- motivational filler, performative praise, or soft language that hides weak work
- drills or quick checks as the learner's first exposure to a skill
- too many strategies for the same small fact set
- written procedures taught as symbol shuffling with no place-value meaning
- pathway rules based on vague country stereotypes instead of concrete learning goals

## Number Bonds, Fact Families, And Similar Tools

These are useful arithmetic tools when used properly.

- Number bonds help the learner see wholes and parts quickly.
- Fact families help the learner connect inverse operations.
- Arrays and equal groups help the learner understand multiplication before relying on chant alone.
- Place-value partitioning helps the learner understand written methods instead of memorising mysterious steps.

These are not fluff by default. They become fluff only when they are stretched into long theory-heavy activities without enough practice.

## Recommended Teaching Sequence For Arithmetic

For most arithmetic topics, use this order:

1. Direct teaching of the idea.
2. One or two worked examples.
3. Short guided practice.
4. Independent practice.
5. Mixed review or quick check.
6. Targeted reteaching of weak patterns.

This keeps the route honest. The learner sees the idea, practises it, and proves it.

## Pathway Design Rules

- Build pathways around readiness and progression, not around school-system labels.
- Keep stages pedagogical. They should describe what the learner is becoming able to do.
- Use age only as entry guidance.
- Make each playlist teachable in a real short session.
- Ensure every playlist contains direct teaching, practice, and checking unless the brief explicitly marks it as review-only or diagnostic.
- Keep the route cumulative. Earlier facts and methods should reappear later.

## Arithmetic-Specific Defaults

### Early fact work

- teach number bonds and fact families directly
- move quickly into mixed recall
- correct weak patterns early instead of letting them harden

### Place value

- make hundreds, tens, and ones explicit
- use exchange as a real trade, not as a secret rule
- do not delay place-value language until after written methods begin

### Written addition and subtraction

- align columns by place value
- explain exchange clearly
- check whether answers are sensible

### Multiplication and division

- teach equal groups, arrays, sharing, and grouping
- connect multiplication and division as inverse operations
- use fact fluency to support larger calculations

## Tone For Adult And Learner Materials

- sound like a strict but sincere parent or teacher
- be warm without becoming indulgent
- be direct without becoming harsh
- name weak work honestly and show the next correction step

## Good Briefing Language

Use prompts such as:

- `Teach this directly.`
- `Use number bonds because they reduce counting and strengthen later written methods.`
- `Keep the explanation short and put the weight on practice.`
- `Do not introduce drills before a lesson note.`
- `Reteach clustered errors immediately.`

Avoid prompts such as:

- `Use the UK style.`
- `Avoid the European style.`
- `Make it more progressive.`
- `Make it more traditional.`

Those labels are too vague to enforce well. State the actual teaching moves instead.
