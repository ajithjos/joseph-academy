# Parent Teaching Note For Written Methods

Do not let written methods hide weak number sense.

## Look for

- whether table facts come quickly enough to support the method
- whether the learner keeps place value aligned
- whether a remainder is treated as an afterthought or explained properly

## Good next moves

- warm up with three table facts before written work
- ask for one estimate before one full calculation
- correct one line at a time instead of rewriting the whole page blindly
