# Multiplication And Division Methods Lesson Note

Multiplication and division methods should still make sense when the numbers get larger.

## Multiplication by partitioning

Example: `23 x 4`

1. Partition `23` into `20 + 3`
2. Multiply each part by `4`
3. `20 x 4 = 80`
4. `3 x 4 = 12`
5. `80 + 12 = 92`

So `23 x 4 = 92`.

## Arrays and scaling

`6 x 14` can be seen as:

- `6` groups of `14`
- an array with `6` rows of `14`
- `14` scaled up `6` times

## Division as sharing or grouping

Example: `84 / 4`

- if `84` is shared into `4` equal groups, each group gets `21`
- if `84` is grouped into sets of `4`, there are `21` groups

So `84 / 4 = 21`.

## Remainders

Example: `97 / 5`

- `5 x 19 = 95`
- `2` are left over
- so `97 / 5 = 19 remainder 2`

The remainder must be smaller than the divisor.

## Check division with multiplication

If `97 / 5 = 19 remainder 2`, then:

- `19 x 5 = 95`
- `95 + 2 = 97`

## Good learner self-talk

- `What does this multiplication represent?`
- `Can I partition the larger number?`
- `Does my remainder make sense?`
- `Can I check this division with multiplication?`

## Quick spoken practice

1. Partition `24 x 3`
2. What does `72 / 8` mean as sharing?
3. Why must a remainder be smaller than the divisor?
