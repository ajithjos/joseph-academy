# Multiplication And Division Methods

This stage teaches the learner to turn known facts into larger written calculations using arrays, partitioning, sharing, and grouping.

## Plain-Language Meaning

Multiplication and division methods should still make sense when the numbers get larger.

## Skills In This Stage

- `interpret_multiplication_as_arrays_and_scaling`
- `multiply_by_one_digit_numbers`
- `interpret_division_as_sharing_and_grouping`
- `divide_by_one_digit_numbers`

## Secure Performance

The learner can solve one-digit multiplication and division calculations with clear method choices, explain remainders, and use multiplication to check division.
