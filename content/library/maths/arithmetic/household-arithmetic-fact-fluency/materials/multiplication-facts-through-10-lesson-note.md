# Multiplication Facts Through 10 Lesson Note

Hard facts are built from easier ones.

## Strategy anchors

- `6 x n = 5 x n + n`
- `7 x n = 5 x n + 2 x n`
- `8 x n = 4 x n + 4 x n`
- `9 x n = 10 x n - n`

Examples:

- `6 x 7 = 35 + 7 = 42`
- `7 x 4 = 20 + 8 = 28`
- `8 x 6 = 24 + 24 = 48`
- `9 x 3 = 30 - 3 = 27`

## Inverse pairs

If `8 x 7 = 56`, then:

- `56 / 8 = 7`
- `56 / 7 = 8`

When a mixed fact looks hard, rebuild it from the anchor instead of guessing.
