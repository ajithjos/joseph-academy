# Readiness And Bonds Within 5 Practice

Use counters, buttons, or fingers if needed, but try to answer from the bond instead of recounting everything.

## Part A: Fill the missing part

1. `1 + __ = 5`
2. `2 + __ = 5`
3. `4 + __ = 5`
4. `5 = 3 + __`
5. `__ + 2 = 5`
6. `5 = 1 + __`

## Part B: Say or write the whole fact

7. `2 + 1 =`
8. `4 - 1 =`
9. `3 + 2 =`
10. `5 - 2 =`
11. `1 + 3 =`
12. `4 - 3 =`

## Part C: Match the family

13. If `2 and 3 make 5`, then `5 - 2 =`
14. If `1 and 4 make 5`, then `5 - 4 =`
15. If `3 and 1 make 4`, then `4 - 1 =`

Coach check: if the learner is still counting every prompt one by one, return to the lesson note and use real objects for one more short session.
