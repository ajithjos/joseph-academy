# Multiplication Facts Foundations Lesson Note

Multiplication means equal groups.

It is not just repeated counting. It is a quick way to describe the same amount again and again.

## Read the groups

- `2 x 4` means `2 groups of 4` or `4 + 4`
- `5 x 3` means `5 groups of 3`
- `10 x 6` means `10 groups of 6`

Ask two questions every time:

- how many groups?
- how many in each group?

## Easy patterns

- `2` facts double the number: `2 x 7 = 14`
- `5` facts end in `0` or `5`: `5 x 8 = 40`
- `10` facts add a zero: `10 x 6 = 60`

## Worked examples

### Example 1: `2 x 7`

- double `7`
- `7 + 7 = 14`
- so `2 x 7 = 14`

### Example 2: `5 x 4`

- `5, 10, 15, 20`
- `5 x 4 = 20`

### Example 3: `10 x 6`

- six tens make `60`
- so `10 x 6 = 60`

## Division partners

If `5 x 4 = 20`, then:

- `20 / 5 = 4`
- `20 / 4 = 5`

If `2 x 9 = 18`, then:

- `18 / 2 = 9`
- `18 / 9 = 2`

## Good learner self-talk

- `I am counting groups, not random jumps.`
- `I know an easy pattern for 2, 5, or 10.`
- `The division fact comes from the same family.`

## Quick spoken practice

1. `2 x 8 =`
2. `5 x 6 =`
3. `10 x 4 =`
4. `20 / 5 =`
5. `14 / 2 =`
6. `30 / 10 =`

Say the multiplication fact and the division partner together until they sound connected.
