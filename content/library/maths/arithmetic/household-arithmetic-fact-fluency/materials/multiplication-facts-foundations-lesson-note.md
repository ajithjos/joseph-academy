# Multiplication Facts Foundations Lesson Note

Multiplication means equal groups.

## Read the groups

- `2 x 4` means `2 groups of 4` or `4 + 4`
- `5 x 3` means `5 groups of 3`
- `10 x 6` means `10 groups of 6`

## Easy patterns

- `2` facts double the number: `2 x 7 = 14`
- `5` facts end in `0` or `5`: `5 x 8 = 40`
- `10` facts add a zero: `10 x 6 = 60`

## Division partners

If `5 x 4 = 20`, then:

- `20 / 5 = 4`
- `20 / 4 = 5`

If `2 x 9 = 18`, then `18 / 2 = 9`.

Say the multiplication fact and the division partner together until they sound connected.
