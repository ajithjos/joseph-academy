# Place Value And Number Sense Check

Keep this brisk and oral where possible.

1. What is the value of the `4` in `438`?
2. Partition `562`.
3. Build a number with `7` hundreds, `2` tens, and `5` ones.
4. Which is greater: `640` or `604`?
5. Which is smaller: `389` or `398`?
6. Put these in order from smallest to greatest: `450`, `405`, `540`.
7. What does the `0` mean in `507`?
8. Say `286` in hundreds, tens, and ones.

Passing guide: 6 or more secure answers means the learner is ready for written addition and subtraction. If place-value language is still weak, repeat this playlist before moving on.
