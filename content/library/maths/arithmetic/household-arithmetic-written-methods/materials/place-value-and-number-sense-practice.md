# Place Value And Number Sense Practice

## Part A: Read and partition

1. Write `462` as hundreds, tens, and ones.
2. Write `580` as `___ + ___ + ___`.
3. What is the value of the `7` in `374`?
4. What is the value of the `2` in `620`?

## Part B: Build the number

5. Build a number with `3` hundreds, `5` tens, and `9` ones.
6. Build a number with `6` hundreds, `0` tens, and `4` ones.
7. Build a number with `9` hundreds, `1` ten, and `8` ones.

## Part C: Compare and order

8. Which is greater: `418` or `481`?
9. Which is greater: `702` or `720`?
10. Order these from smallest to greatest: `356`, `365`, `336`.
11. Order these from greatest to smallest: `804`, `840`, `480`.

## Part D: Explain

12. Why is `530` greater than `503`?
13. Why is `609` less than `690`?

Coach check: if the learner compares the last digit first, stop and return to the rule `compare the biggest place first`.
