# Written Addition And Subtraction Check

Do not help during the check except to read the question.

1. `58 + 27 =`
2. `346 + 178 =`
3. `92 - 36 =`
4. `503 - 267 =`
5. Why do you exchange in `58 + 27`?
6. Why do you exchange in `92 - 36`?
7. Which answer is more sensible for `346 + 178`: `524` or `421`?
8. Which answer is more sensible for `503 - 267`: `236` or `770`?

Passing guide: 6 or more secure answers means the learner is ready for multiplication and division methods. If written answers are right but explanations are empty, repeat one more guided session before moving on.
