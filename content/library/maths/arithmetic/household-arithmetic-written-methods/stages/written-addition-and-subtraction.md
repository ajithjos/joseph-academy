# Written Addition And Subtraction

This stage teaches the learner to align digits by place value and exchange deliberately when needed.

## Plain-Language Meaning

Written methods are a clear way to organise place-value work on paper. They are not magic steps to memorise blindly.

## Skills In This Stage

- `add_two_and_three_digit_numbers`
- `subtract_two_and_three_digit_numbers`
- `explain_exchange_in_written_methods`

## Secure Performance

The learner can complete two-digit and three-digit addition and subtraction accurately, explain each exchange, and notice when an answer is not sensible.
