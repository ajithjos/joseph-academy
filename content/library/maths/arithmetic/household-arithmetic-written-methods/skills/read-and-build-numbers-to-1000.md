# Read And Build Numbers To 1000

- successful performance: reads, writes, and builds numbers through `1000` using hundreds, tens, and ones language
- likely mistakes: reads each digit separately or confuses the spoken number with the digit order
- out of scope: decimals and numbers above `1000`
