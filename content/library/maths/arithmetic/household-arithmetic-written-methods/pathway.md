# Household Arithmetic Written Methods

This pathway teaches the arithmetic structure that sits on top of fact fluency.

It is for learners who now need to understand place value, written addition and subtraction, and the first useful multiplication and division methods. For many learners, this is the pathway that turns arithmetic from a pile of separate tricks into one connected system.

Use this pathway alongside or after [Household Arithmetic Fact Fluency](../household-arithmetic-fact-fluency/pathway.md). Facts still matter here. The written methods become much easier when the fact layer is secure.

## Why This Pathway Exists

Many learners can copy a written method or finish a worksheet without really understanding what the digits are doing.

Typical problems look like this:

- digits are lined up by appearance instead of by place value
- exchange in addition or subtraction is memorised as a rule but cannot be explained
- multiplication and division methods feel random because equal groups and scaling are not clear enough
- the learner gets an answer but cannot tell whether it is sensible

The job of this pathway is to make the structure of arithmetic visible and usable.

## What Good Progress Looks Like

At the end of this route, the learner should be able to:

- read and build numbers through `1000` with clear hundreds, tens, and ones language
- align written addition and subtraction correctly and exchange when needed
- explain why an exchange happened instead of only copying the step
- use multiplication and division structure for one-digit written methods and remainder problems
- check whether an answer is sensible before moving on

## Daily Delivery Shape

- session length: 12 to 18 minutes
- cadence: 4 sessions per week
- rhythm: one teach-and-model session, two practice sessions, and one check or review session

Meaning first still matters. If the learner cannot explain what the digits represent, stop and rebuild the idea instead of adding more pages.

## The Route

### 1. [Place Value And Number Sense](./playlists/place-value-and-number-sense.md)

Start here for most seven-year-olds and for any older learner who still confuses digit value with place value.

What changes here:

- numbers stop looking like a string of separate digits
- the learner can talk about hundreds, tens, and ones clearly
- comparison and ordering become easier because the learner knows which digit matters first

### 2. [Written Addition And Subtraction](./playlists/written-addition-and-subtraction.md)

This stage turns place-value understanding into usable written methods.

What changes here:

- columns are aligned by place value instead of by spacing guesswork
- exchange becomes a meaningful trade, not a hidden ritual
- the learner can check an answer for reasonableness before moving on

### 3. [Multiplication And Division Methods](./playlists/multiplication-and-division-methods.md)

This stage turns known facts into larger calculations.

What changes here:

- multiplication is treated as scaling, arrays, and partitioning
- division is treated as sharing and grouping with a remainder that means something
- the learner starts using multiplication to check division and vice versa

## Entry Guidance By Approximate Age

### Seven-Year-Old

Start at [Place Value And Number Sense](./playlists/place-value-and-number-sense.md).

Do not rush into formal written methods if hundreds, tens, and ones are still vague.

### Ten-Year-Old

Start with a quick baseline from [Written Addition And Subtraction](./playlists/written-addition-and-subtraction.md).

If written addition and subtraction are already solid, move into [Multiplication And Division Methods](./playlists/multiplication-and-division-methods.md). If place value language is weak, route back to [Place Value And Number Sense](./playlists/place-value-and-number-sense.md) immediately.

## Exit Conditions

This pathway is doing its job when the learner can:

- explain place value through `1000`
- add and subtract in columns with sensible exchange
- use one-digit multiplication and division methods without treating the page as a guessing exercise
- check work for reasonableness using the size of the numbers involved

This is not the end of all arithmetic. It is the point where the learner has a stable written-methods base to build on.
