# Multiplication Facts Through 10 Lesson Note

Hard facts are built from easier ones.

You do not need to guess at `6`, `7`, `8`, and `9` facts. Build them from facts you already trust.

## Strategy anchors

- `6 x n = 5 x n + n`
- `7 x n = 5 x n + 2 x n`
- `8 x n = 4 x n + 4 x n`
- `9 x n = 10 x n - n`

## Worked examples

### Example 1: `6 x 7`

- start with `5 x 7 = 35`
- add one more `7`
- `35 + 7 = 42`

### Example 2: `7 x 4`

- `5 x 4 = 20`
- `2 x 4 = 8`
- `20 + 8 = 28`

### Example 3: `8 x 6`

- `4 x 6 = 24`
- double `24`
- `24 + 24 = 48`

### Example 4: `9 x 3`

- `10 x 3 = 30`
- subtract one `3`
- `30 - 3 = 27`

## Inverse pairs

If `8 x 7 = 56`, then:

- `56 / 8 = 7`
- `56 / 7 = 8`

If `9 x 6 = 54`, then:

- `54 / 9 = 6`
- `54 / 6 = 9`

## Good learner self-talk

- `I can rebuild this from an easier fact.`
- `Nine times means ten times minus one group.`
- `Division tells the same story backwards.`

When a mixed fact looks hard, rebuild it from the anchor instead of guessing.
