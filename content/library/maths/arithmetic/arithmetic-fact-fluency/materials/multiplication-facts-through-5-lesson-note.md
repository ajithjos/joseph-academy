# Multiplication Facts Through 5 Lesson Note

Build the harder facts from anchors you already trust.

You already know helpful facts from `2`, `5`, and `10`. Use them to build `3` and `4` facts instead of starting from nothing.

## Use `2` facts to build `3` and `4`

- `3 x 4 = 2 x 4 + 1 x 4 = 8 + 4 = 12`
- `3 x 5 = 2 x 5 + 1 x 5 = 10 + 5 = 15`
- `4 x 3 = 2 x 3 + 2 x 3 = 6 + 6 = 12`
- `4 x 5 = 2 x 5 + 2 x 5 = 10 + 10 = 20`

## Worked examples

### Example 1: `3 x 4`

- start with `2 x 4 = 8`
- add one more `4`
- `8 + 4 = 12`

### Example 2: `4 x 3`

- double `3` to get `6`
- double `6` to get `12`
- so `4 x 3 = 12`

## Keep the set small and secure

- `3 x 2 = 6`
- `3 x 3 = 9`
- `3 x 4 = 12`
- `4 x 2 = 8`
- `4 x 3 = 12`
- `4 x 4 = 16`

## Good learner self-talk

- `I know an easier fact inside this one.`
- `Three groups means two groups and one more group.`
- `Four groups can be two doubles.`

## When the facts are mixed

Do not recite the whole table.

Look at the factor, choose the anchor, and rebuild the fact.
